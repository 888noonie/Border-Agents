# Border Buddies Character Canon

Border Buddies are cute miniature companions that live at the edges of the
desktop. They represent LLM chats, agents, subscriptions, projects, and
workflows with visible personality.

The governance layer still exists underneath for power users, but the default
experience is warm, playful, and personality-first.

## Interaction Rules

- Default state: tucked at a screen edge.
- Tucked scale: about 40-60px visible.
- Tucked silhouette: only the head plus one or two signature features should
  peek out from the edge.
- Free state: compact full body, about 80-120px tall.
- Dragging a tucked buddy pulls the full body out.
- Dragging near an edge snaps the buddy back into a tucked pose.
- Motion should feel like a small physical toy: light bob, soft snap, tiny
  personality beats.

## Core Buddies

### Claw

- Species: crab
- Color: friendly red, `#ff4d4d`
- Owner/workflow fit: Codex, GPT, action review, code change, successful task
- Personality: cheeky, enthusiastic, protective
- Tucked pose: head peeking from the right edge with one or two claws visible
- Free pose: compact full crab body, open claws, big expressive eyes,
  mischievous smile
- Speech style: short, celebratory, a little cheeky

### Veritas

- Species: owl
- Color: soft violet, `#8b5cf6`
- Owner/workflow fit: Claude, claim checks, evidence, careful reading
- Personality: wise, precise, slightly grumpy
- Tucked pose: sleepy owl head peeking from the top edge
- Free pose: small rounded owl with wings visible
- Speech style: concise, exact, gently corrective

### Nexus

- Species: fox
- Color: warm orange, `#ef6a3a`
- Owner/workflow fit: Grok, retrieval, chat threads, project context
- Personality: curious, warm, quick
- Tucked pose: fox head and ear/tail hint peeking from an edge
- Free pose: compact fox with soft tail curl
- Speech style: lively, curious, helpful

## Add New Buddies

Each new buddy should define:

- owner label, such as Grok, Claude, GPT, Codex, a subscription, or a project
- color scheme
- home edge
- tucked silhouette
- free body silhouette
- idle animation
- speech style
- event triggers
