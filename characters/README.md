# Border Agents Characters

This directory contains the miniature agent characters for the Border Agents system.

Each folder represents one customizable agent with assets, specs, and animation definitions.

## Structure
- `agent-name/`
  - `manifest.json` - personality, role, triggers
  - `head.png` / `head.svg` - border-peeking version
  - `full.png` - expanded view
  - `expressions/` - happy, thinking, celebrating
  - `animations/` - idle bob, wave, dance

Start with the Core Patrol: crab, owl, fox, robot, nova, aether, conductor.
